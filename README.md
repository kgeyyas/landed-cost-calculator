## 🧭 Overview

**netto** is an open-source landed cost calculator designed for importers, exporters, sourcing professionals, and global trade teams.

It helps estimate the true cost of importing products by combining product value, freight, customs duties, taxes, insurance, and other import-related expenses into a transparent and structured calculation.

Incomplete landed cost estimates can lead to:

- Unexpected import expenses
- Incorrect pricing strategies
- Reduced profit margins
- Poor sourcing decisions

Netto is designed to reduce these risks by providing clear, explainable, and data-driven landed cost calculations.

### 🧠 System Design Philosophy

Netto is designed around three core principles:

- Transparency over hidden cost calculations
- Practical sourcing decisions over theoretical estimates
- Visibility across the complete import cost structure

The goal is not simply calculating landed cost, but helping businesses understand where costs originate and how procurement decisions impact profitability.
